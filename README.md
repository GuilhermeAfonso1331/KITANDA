# KITANDA – Sistema Inteligente de Reservas para Restaurante

O **KITANDA** é um sistema de reservas e gestão de mesas desenvolvido no âmbito da unidade curricular de **Projeto Aplicado de Software e Aplicações I**.  
O objetivo principal é permitir que um restaurante faça reservas, acompanhe o estado das mesas em tempo real, personalize sugestões para clientes regulares e utilize algoritmos simples para melhorar a experiência.

Este repositório contém:
- Protótipo do sistema
- Código-fonte em Java
- Backlog do produto (Features e User Stories)
- Planeamento de Sprints

---

## Objetivos do Projeto

- Permitir reservas online de forma simples e intuitiva  
- Sugerir automaticamente mesas premium para clientes regulares  
- Fornecer ao gestor uma visão em tempo real da ocupação do restaurante  
- Implementar um sistema básico de fidelização (pontos e recompensas)  
- Criar um mapa visual das mesas para clientes e staff  
- Garantir um planeamento estruturado do desenvolvimento através de Sprints
